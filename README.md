# chayka.me

Personal website (resume + blog + projects) built with:

- Markdown
- Vue 3
- TypeScript
- CSS

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The project is configured for GitHub Pages project deployment with base path:

`/chayka.me/`

If you deploy to a custom domain root, change `base` in `vite.config.ts`.
